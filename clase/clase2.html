El CSS

Css es un lenguaje de estileaje que complementa a Html para Rendelizar contenido con Estilo. Se puede:
  • Invocar desde Dentro
  • Invocar externa con un link
  • A traves de un Atributo No es Recomendable ya que se va aun objeto

Ejemplo tipico de una hoja Css: Body { Background: #ffffff; }

Ejemplo tipico de una hoja Css:
  • Poner Id en la etiqueta < id=""> , se debe llamar en el css a traves de #
  • el atributo class , en la pagina css se llama a traves del . antes del nombre del nombre de la clase
  • Si la invocacion es general como Bodi no hace falta nada
  • Seleciionar por atributo: Si por ejemplo en una etiqueta tenemos un atributo Alt por ejemplo. y en la pagina Css esta [alt] entre corchetes todas las etiquetas que tengan alt sea cual sea tendran esas caracteristicas
    [alt]{lo que sea}
    [title="parrafo"]{Lo que sea}
