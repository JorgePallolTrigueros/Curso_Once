TEXTO

Las propiedades mas usadas:
  • color: Hereditaria si el padre la tiene, por ejemplo un div h1 lo tendra
  • Font-family: A traves de texto, se llama al tipo de letra "Open Sans";
  • Font Size: Define el tamaño y es hereditaria
  • text-decoration:none -->Quitar o poner algo de decoracion en el texto
  • text-shadow(x,y,r,c): x y (Cordenadas) r (Radio de difusion) C (color de la sombra)
  • Text-align:Alineamiento del Texto: -->Solo a bloques y las posibilidades son
    • Left
    • Right
    • Justify
    • Center
  • Line-height: La altura de lineas. "El Interlineado". Es hereditario
  • letter-spacing Espacio entre lineas, se puede usar vlaores negativos
  • Word-spacing Espacio entre palabras, se puede usar vlaores negativos

DIV (CAJITAS)

Las propiedades mas usadas:
  • Whidth: La anchura de la caja, se puede definir con medidas
  • height: La altura de la caja, se puede definir con medidas
  • Padding: Define el Margen interno
    • padding-left
    • padding-Right
    • padding-Top
    • padding-botton
    • top right botton left
  • Margin Separacion entre cajas o con los elementos que le rodean, tiene los mismos lados que el div
  • Border Sus parametros son
    • Ancho
    • Style
    • Color
    • Box-sizing:Border-box
    • Box-sizing:Content-box
